My Projects

Myanmar InnoHub using Django

Myanmar InnoHub is developed to be a central portal for all type of technogolical innovations. Aimed to allow users to post, share and display their own projects on thier profile. Also contains project browsing, filter/search, and feedback features.

POS System using Laravel

Being my first Laravel project built from scratch, this project explores most commonly used features of the Laravel architecutre. This includes routing, middleware, authentication control, ORM, and API.

UCSY E-Library using J2EE

UCSY E-Library is the initial version of digital catalog management for UCSY Library. It currently has features for browsing and reading ebooks online. It also has librarian features for catalog management and admin operations.

Design Modeling for Treasure's Hub

This project reflects the true modeling power of UML in software engineering. Including both static and dynamic aspects of the system, it fully represents almost all possible perspectives of software modeling. Key models in this project are use case model, state diagram, and design class models.

FitLife Dataset Analysis

FitLife Dataset Analysis project is one of my first data analytics projects done as part of the UCSY coursework. Starting from dataset selection and preparation, it went through all data analysis steps producing both descriptive knowledge and predictive models. Frequent pattern mining and predictive model training are also done as experiment beyond the curriculum limits.

MLIP Database Project

MLIP means Ministry of Labor, Immigration and Population of Myanmar. This project tries to create a central repository for storing countrywide data of citizens in digital format. It includes precise ER modeling work further implemented with MySQL.

Smart Staircase using Arduino

Automated lightning system for the staircases that enhances safety, reduces energy consumption, and offers user-friendliness. By utilizing motion and light sensors, the project aims to enable staircases lights only when needed, improving safety by ensuring proper visibility while conserving energy.

Online Fresh Fuits Store Modeling

This project is demonstration of the first half of software engineering concepts taught in UCSY. It solves the business problems with effective and precise requirement engineering, and proceeds to model the requirements with relevant models. This is the basis for futher design modeling.

Landing Page for the Crown Series

This project is my very first web project with only HTML, CSS, and JS. It is a movie review site for the English series named 'The Crown', featuring the casts, plots, and scenes.

Ecommerce System Database

Being the second database project, this project reflects the advanced parts of database technology. It includes all the ER modeling, MySQL implementations, and also has procedures, triggers, functions, and indexing features.
